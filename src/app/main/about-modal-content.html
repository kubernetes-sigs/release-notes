This tool is designed to view the official Kubernetes release notes within the browser. The note contents are directly provided via the Kubernetes SIG release team and are transformed per release into JSON format to be consumable by this application. The release notes can be filtered by release version, kind, area, SIG or documentation. It is also possible to do full text search by querying the Search input box at the top of the page.

Do you want to contribute or give further feedback? Great, feel free to open up a pull request or issue in the GitHub repository. Beside this, you can always contact us via the official Kubernetes Slack #sig-release channel.
